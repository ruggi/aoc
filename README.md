# aoc - advent of code

Very _very_ **_very_** ugly and utilitarian [Advent of Code](https://adventofcode.com) solutions.

```plain
NAME:
   aoc - advent of code solutions

USAGE:
   aoc [global options]

GLOBAL OPTIONS:
   --year int, -y int
   --day int, -d int
   --input string, -i string
   --part int, -p int         (default: -1)
   --help, -h                 show help
```

## Run this thing

```sh
go run main.go -y 2025 -d 1 -i puzzles/2025/day1.txt
```
